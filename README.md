# Overview
This repo contains the code for this <a href="https://ieeexplore.ieee.org/document/9464614">paper</a> which describes my contribution to <a href="https://www.just.edu.jo/icics/icics2021/mowjaz/Task%20Description.html">ICICS 2021 Mowjaz Multi-Topic Labelling Task</a>. Mowjaz is an Arabic topical content aggregation mobile application for news, sport, entertainment and other topics from top publishers that users can follow. The paper describes the approach to classify articles using Bi-directional Gated Recurrent Unit (Bi-GRU) with AraVec embeddings. The F1-score of this system is 0.8344 which shows a significant improvement over the baseline models, and resulted in ranking 6th place.
